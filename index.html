How To Use
Quick reference guide for the RCM Dashboard
⚡ Live Status Board
→ Shows who is assigned where today across all 10 offices and all RCM categories.
→ Drag any staff token from the roster or another cell into any grid cell to assign them.
→ Right-click a token in a cell to send them to Lunch, Away/Off, or remove them from that spot.
→ Cells with a pink background have an active Help Request in that spot.
→ Ghost text (faint names) shows what's on the Assignment Board as a reference behind the live view.
→ OWN TEAM ghost text appears on the right side of a cell — it means that office handles it in-house.
→ Changes sync automatically within seconds — anyone viewing the dashboard will see updates in real time.
📋 Assignment Board 🔒 Admin Only
→ This is the permanent reference layout — the default "who covers what" for each office and category.
→ Editing requires the manager PIN. Click Unlock to Edit in the board header, enter the PIN, then drag tokens to set assignments.
→ Drag tokens here to set standing assignments. These show as ghost names on the Live Board.
→ Use OWN TEAM (the dark token) to mark that an office handles a category themselves.
→ Click ↺ Reset Defaults (visible when unlocked) to restore the original assignment layout if needed.
→ You can add or rename columns using the pencil icon on any column header, and add new offices with the + button (admin only).
👥 Staff Roster 🔒 Admin Only
→ Lists all active staff with their role, department, manager, and responsibilities.
→ Unlock to Edit requires the manager PIN. Once unlocked you can add staff, edit profiles, and manage departments.
→ To remove a staff member, open their profile and click Move to Inactive. Choose Terminated or Inactive as the reason — they move to the Inactive department instead of being deleted.
→ Drag any staff token from the roster panel directly onto the Live or Assignment boards to place them.
→ Management always sorts to the top; Inactive staff always sort to the bottom (shown in gray).
🚨 Help Requests
→ Click 🚨 Need Help in the header to create a help request. Fill in what's needed and who's requesting.
→ Drag the red help token from the panel into a grid cell on the Live Board to show which office/category needs help.
→ Click the pulsing red token anytime to open it — you can add chat messages, mark helpers, update the checklist, and add notes.
→ Click Mark Helped when someone steps in — the token turns green. Click Resolve & Remove when the issue is fully resolved — it archives to Help Records.
→ All resolved requests are saved in the Help Records tab with the full chat history and details.
☁ Saving & Syncing
→ Every change saves automatically — you don't need to press anything. Look for ☁ Synced in the header to confirm.
→ The dashboard checks for updates from other locations every 15 seconds and refreshes automatically.
→ The Publish button in the top-right forces an immediate sync — use it only if you need to push changes right away.
→ Publish requires the manager PIN. Day-to-day, auto-save handles everything without it.
